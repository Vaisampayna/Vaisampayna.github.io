About Me

I'm a Master's student at the Department of Computer Science & Engineering, Indian Institute of Technology, Kharagpur.

My research focuses on Secure Multi-Party Computation (MPC) and its applications in privacy-preserving technologies. I work on building efficient cryptographic protocols that allow multiple parties to jointly compute functions on their private data—without revealing anything beyond the final result.

I'm passionate about bridging the gap between theoretical cryptography and practical systems, from implementing OLE protocols over prime fields to building end-to-end encrypted mobile applications.

Secure MPC Android App

A two-party secure computation Android application that enables two users to jointly compute statistical functions (mean, variance, std dev, median, mode) over their private data using WebSocket-based relay communication.

Kotlin Jetpack Compose WebSocket Node.js
View on GitHub →

OLE from Garbled Circuits

Implementation of Oblivious Linear Function Evaluation using Yao's garbled circuits with Free-XOR and point-and-permute optimizations. Benchmarked with NP-OT and IKNP-OTE over LAN, achieving 10× speedup with OT extension.

C OpenSSL EMP-toolkit CBMC-GC-2

Batch OLE from Noisy Encoding

Efficient batch OLE protocol from IPS09, using polynomial interpolation and noisy linear encodings. Scaled to 2²³ OLEs over 64-bit prime fields with preprocessing-based OT extension for minimal online overhead.

C/C++ FLINT EMP-toolkit

Privacy-Preserving Statistics Interface

Terminal-based MPC system supporting 5 statistical functions using three distinct cryptographic paradigms: additive secret sharing, Yao's garbled circuits, and semi-honest 2PC via EMP toolkit.

C/C++ EMP-toolkit Networking
